Personal-Todo

my personal todo list with local storrage caching and some usable functionalities

About this project

Ich habe dieses Projekt im Rahmen meiner Weiterildung zum Webentwickler erstellt. Man kann persönliche Listen erstellen und in diese die todo tasks schreiben. Die Todo Tasks kann man als erledigt markieren und anschließend löschen. Außerdem kann man zwischen den verschiedenen Listen wechseln, dadurch bleibt stets die Übersicht erhalten.
