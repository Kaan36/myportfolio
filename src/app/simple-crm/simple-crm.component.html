Simple-CRM

This is a CRM-project to organize the personal in the system and save the personal values

About this project

Ich habe mit diesem Projekt während meiner Einarbeitung beim Angular-Framework angefangen. Ich habe in diesem Projekt die Aufgabe übernommen einem Hotel die Übersicht zugewähren über den Status der belegten sowie freien Zimmer intern. Mit dieser Applikation hat man immer stets die Übersicht und kann einfach den Buchungsstatus mit den Daten der Kunden eintragen.
