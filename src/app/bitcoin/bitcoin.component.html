Bitcoin

Eine Bitcoin Wep-Applikation die in Verbindung mit einer API immer die aktuellsten Daten und Trends darstellt.

About this project

Ich habe mit diesem Projekt während meiner Weiterbildung zum Frontend Webentwickler angefangen. Meine Aufgabe sowie Ziel war es eine Darstellung des aktuellen Bitcoinwertes darzustellen. Da mehr Infos in der API beinhaltet waren habe ich mich dazu entschieden einen Bitcoin-exchange-rate zu bauen. Man kann den aktuellen Bitcoin Wert mit einer individuell eingegeben Dollar Menge berechnen. Außerdem kann man den Trendverlauf des Bitcoins über die letzten Jahre Grafisch darstellen.
