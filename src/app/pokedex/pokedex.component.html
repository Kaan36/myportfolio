Pokedex

Ein Pokedex Projekt, welches mithilfe einer API alle Pokemon in einem einzigartigen Design darstellt.

About this project

Ich habe mit diesem Projekt während meiner Weiterbildung zum Frontend Webentwickler angefangen. Meine Aufgabe sowie Ziel war es eine Visuelle Darstellung der Pokemons mithilfe einer API herzustellen. Diese Applikation stellt durch einen asynchronen Request eine Verbindung mit der Pokemon API und anschließend werden die im Objekt erhaltenen Informationen in mein erstelltes Interface eingebettet. Jegliche Informationen sowie Bilder sind von der API-Schnittstelle gefetcht.
