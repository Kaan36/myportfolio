TicTacToe

Ein simples TicTacToe spiel mit VanillaJs erstellt

About this project

Ich habe mit diesem Projekt während meiner Weiterbildung zum Frontend Webentwickler angefangen. Meine Aufgabe sowie Ziel war es ein simples Spiel zu entwickeln um mich mit den Möglichkeiten von CSS und Javascript vertraut zumachen. Die Symbole wurden mit purem CSS erstellt. Es ist ein schlichtes Design ausgewählt worden welches zugleich auch responsiv ist.
